WAF(ModSecurity)는 무엇을 하나요?

WAF(ModSecurity) 는 웹 서버(Apache) 앞단에서 HTTP 요청의 내용 을 검사해 웹 공격(SQLi/XSS/LFI 등)을 막는다. 방화벽이 IP/포트, IPS 가 패킷 패턴을 본다면, WAF 는 HTTP 의미(파라미터·헤더·본문)를 이해하고 차단한다.

  • SecRule — 변수(ARGS/REQUEST_URI…) + 연산자(@rx/@contains/@pm) + 액션(deny/pass…)
  • OWASP CRS — 미리 만들어진 표준 룰셋(3.3.2). 941=XSS, 942=SQLi …
  • 이 콘솔의 룰은 적용 전 configtest 로 문법을 검사해 Apache 를 보호한다.
